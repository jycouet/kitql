## Install

```bash
npm install @kitql/eslint-config --D
```

### eslint config

`.eslintrc.js`

```js
/** @type { import("eslint").Linter.FlatConfig } */
module.exports = {
  extends: ['@kitql'],
  rules: {}
}
```

### prettier config

`.prettierrc.cjs`

```js
const config = require('@kitql/eslint-config/.prettierrc.cjs')

module.exports = {
  ...config
  // Some custom things?
}
```

